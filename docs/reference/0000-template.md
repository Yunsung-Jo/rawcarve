# [제목]

- **출처:** 공식 스펙(링크·§) | 실측(방법·표본) | 외부 문서(링크) — 섞이면 모두 기재
- **최종 수정:** YYYY-MM-DD  *(살아있는 문서 — 지식이 늘면 갱신·확장한다)*
- **범위(선택):** 무엇을 담고 무엇은 다른 문서에 두는지 한 문장 — 머리말 또는 본문 어디든.
- **관련(선택):** 코드(`carver/...`)·다른 reference·ADR 링크 — 본문에 인라인해도 된다.

---

(내용은 자유 형식. 스펙 사실과 프로젝트 실측이 섞이면 실측 구간을 `(실측)`으로 표시한다.
모델 지식으로 적는 항목은 권위 출처·코드와 대조해 검증한 뒤 그 출처로 표기한다.)
